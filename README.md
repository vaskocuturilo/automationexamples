# automation scripts examples for http://the-internet.herokuapp.com/

You will need the following technologies available to try it out:
* Git
* Maven 3+
* JDK 8
* IDE of your choice 
* Web browser Chrome

### Ho w to run

```mvn clean test```


### Generate Allure report 

```mvn allure:report```

### Open Allure report in browser

```mvn allure:serve```
